✅ 사용 방법 안내
1. 기기에 맞는 프로그램을 다운로드 및 실행
2. 화면 왼쪽 위에 나타난 숫자(ID)를 알려주세요.
⚠️ 다운로드가 자동으로 시작되지 않거나 차단될 경우: 1. 키보드에서 Ctrl + J를 눌러 [다운로드 목록]을 엽니다.
2. 목록에서 해당 파일을 찾아 [계속] 또는 [위험 수용]을 선택한 뒤,
3. 파일을 직접 PC의 바탕화면에 저장하여 실행해 주세요.

⚠️ 실행 시 경고창이 뜨면 '추가 정보'를 누른 뒤 '실행'을 선택해 주세요.
